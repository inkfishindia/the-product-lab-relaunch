## B2B PIPELINE TRACKER (Notion Database Schema)

```
Properties:
- Company Name (Title)
- Contact Person (Text): Decision maker
- Deal Value (Number)
- Stage (Select): Lead / Qualified / Proposal / Negotiation / Closed
- Probability (Select): 25% / 50% / 75% / 90%
- Expected Close (Date)
- Last Contact (Date)
- Next Action (Text)
- Industry (Select): Vertical classification
- Deal Source (Select)
- Notes (Text): Conversation history
- Documents (Files): Proposals, contracts
- Revenue Impact (Formula): Value × Probability
```

**Pipeline Automation (N8N):**
- Trigger: Every Tuesday and Friday
- Actions: Review stage progression, identify stale opps (>7 days), generate follow-ups, calculate weighted value, send summary to Dan, alert for approaching close dates

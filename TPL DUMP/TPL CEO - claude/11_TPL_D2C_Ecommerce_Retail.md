### WooCommerce Configuration
**Core Plugins:**
- WooCommerce Subscriptions (artist drops)
- WooCommerce Bookings (workshops)
- WooCommerce Product Add-Ons (customization)
- WooCommerce Quick View
- WooCommerce Wishlist

**Performance:**
- WP Rocket (caching)
- Smush (image optimization)
- WP-Optimize (database cleanup)

**SEO & Marketing:**
- Yoast SEO Premium
- MonsterInsights (analytics)
- OptinMonster (email capture)

### Mobile-First Responsive Design
```
Breakpoints:
- Mobile: 320-768px (70% traffic) — Priority
- Tablet: 768-1024px (15%)
- Desktop: 1024px+ (15%)

Requirements:
- Touch-friendly buttons (44px minimum)
- Thumb-zone navigation
- Swipe gestures for product galleries
- One-thumb checkout process
- Progressive web app capabilities
- Page load: <3 seconds
```

## Key Prompt Patterns Used

### Expert Advisory Panel
- Moderated by real agencies (Mother London, McKinsey, Collins)
- XML-structured outputs
- Persona-based thinking partners
- Revenue-impact prioritization filter

### Working Modes in Prompts
```
/flow   — deep creative work, no interruptions
/founder — strategic decisions, ops management, financial reasoning
/reflect — introspection, journaling, pattern recognition
/mentor — coaching, teaching, knowledge transfer
/build  — technical architecture, system design, coding
```

### 1-3-1 Decision Framework
Used across multiple prompts for structured decision-making:
1. **1 Problem** — Clear statement of the challenge
2. **3 Options** — Viable solutions with trade-offs
3. **1 Recommendation** — Best path forward with rationale

# The Product Lab (TPL) — Strategic Goals & Execution Plan

## 5-Year North-Star Goals (2025–2030)

| Area | Target |
|---|---|
| **Marketplace B2C** | 10,000 active artists + ₹150 Cr in gross merchandise value (GMV) |
| **B2B & White Label** | 40% of revenue from retainer-based or subscription merch plans |
| **Retail Footprint** | 5 El Patio hubs + 100 licensed/franchised kiosks in Tier 1 & Tier 2 cities |
| **Tech Edge** | Launch AI semantic search, real-time mobile dashboard, Web3 merch authentication layer |
| **Sustainability** | 50% of the catalog in recycled or low-impact fabrics and packaging |

---

## Immediate Execution Levers (Next 12 Months)

| Lever | Action |
|---|---|
| **Scale B2B Revenue** | Target high-cash flow corporate clients, retainer plans, and repeat agency partners |
| **Automate Core Ops** | Sync ERP with Airtable for live production tracking and auto-order processing |
| **Launch Signature Collection** | Drop 35 exclusive SKUs for the TPL in-house range; leverage pre-sale model |
| **Redesign Marketplace UX** | Implement semantic search, personalized artist feeds, and PWA interface |
| **Franchise Micro-Retail Kit** | Develop plug-and-play retail kits for licensing to cafes, venues, and creators |
| **El Patio Pilot Rollout** | Secure strategic sponsors & partnerships to soft-launch the first experiential hub |

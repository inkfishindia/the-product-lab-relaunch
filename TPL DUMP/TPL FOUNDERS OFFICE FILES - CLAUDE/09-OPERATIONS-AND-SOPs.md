## Named Operational Frameworks

### FRESH Store Inspection Method
- **F — Facilities**: All systems functioning, clean, ready
- **R — Retail Space**: Displays perfect, signage correct, pathways clear
- **E — Equipment**: POS, music, lighting operational
- **S — Staff**: Properly dressed, positioned, prepared
- **H — Human Experience**: Store ready to deliver exceptional service

### DELIGHT Customer Service Protocol
| Step | Action |
|------|--------|
| **D — Discover** | Greet customer within 10 seconds |
| **E — Engage** | Initiate appropriate conversation |
| **L — Listen** | Focus on customer needs |
| **I — Inspire** | Share relevant product/artist stories |
| **G — Guide** | Help with product selection |
| **H — Harvest** | Gather feedback/contact info |
| **T — Thank** | Create memorable farewell |

### Service Recovery Process
1. **Acknowledge** the issue without defensiveness
2. **Apologize** sincerely
3. **Resolve** to customer satisfaction
4. **Thank** them for bringing it up
5. **Document** for process improvement
6. **Follow up** when appropriate

## Daily Operational Excellence Checklist

### Morning Preparation (9:00–10:00 AM)
- [ ] Complete store opening (FRESH inspection)
- [ ] Verify zones are merchandised and restocked
- [ ] Check lighting, music, environment
- [ ] Ensure cash float in register
- [ ] Brief team on daily goals and promotions
- [ ] Count top 20 selling accessories → update Airtable
- [ ] Review overnight online orders
- [ ] Check website functionality
- [ ] 15-minute morning huddle with all staff

### Mid-Day Operations (10:00 AM–4:00 PM)
- [ ] 10:00 AM: Process pending online orders batch
- [ ] Respond to all customer emails within 4 hours
- [ ] 1:00 PM: Replenish displays, tidy zones, update sales tracking
- [ ] 2:00 PM: Process new online orders batch
- [ ] 3:00 PM: B2B follow-up (active orders, proposals, pipeline)

### Evening Operations (4:00 PM–8:00 PM)
- [ ] 5:00 PM: Final order processing batch
- [ ] 7:00 PM: Begin closing preparation, restock, pre-count cash
- [ ] 7:30 PM: Analytics review (sales, inventory, website traffic)
- [ ] 8:00 PM: Complete store closing procedure

### Daily Kaizen
- Morning: Identify one operational friction point
- Evening: Evaluate progress, document changes, select next-day focus

## Order Processing Workflow

### Multi-Channel Order Intake
| Channel | Process |
|---------|---------|
| Website (WooCommerce) | Auto-capture → Transfer to Softr/Airtable → Production docs → Confirmation |
| WhatsApp | Collect details → Create formal order in Softr → Confirm specs → Production |
| Instagram DMs | Transfer to order system → Follow up for missing info → Create formal order |
| B2B/Offline | Process in Airtable "Offline Orders" → Assign ID (IF-OF-XXXXX) → Production |

### Order Status Flow
```
Lead → Confirmed → Production → Processing → Packed → Shipped → Delivered
```
Also: Escalated, On Hold, Cancelled

### Daily Order Batches 🔢
- 10:00 AM — First batch
- 2:00 PM — Second batch
- 5:00 PM — Final batch

## Visual Merchandising Principles

1. Storytelling: Each display tells a cohesive story
2. Elevation: Key products at eye level (5'6" center point)
3. Rule of Three: Products in groups of three
4. Color Blocking: Harmonious color progression
5. Negative Space: Breathing room between products
6. Cross-Category: Mix complementary products
7. Rotation: 25% of displays weekly, full reset monthly

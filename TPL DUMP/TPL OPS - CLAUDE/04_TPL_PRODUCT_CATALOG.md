## DETAILED PRODUCT SPECIFICATIONS

### ACCESSORIES

#### KEYCHAINS

| Attribute | Specification |
|-----------|--------------|
| **Materials** | Fiber Print OR Birch Wood |
| **Print Type** | Single side or Double side |
| **Attachment** | Split Ring (Metal) — sizes 15mm, 20mm, 25mm, 30mm |
| **Tool Required** | Pliers |
| **Assembly Complexity** | Simple |
| **Assembly Time Target** | 1–2 minutes per unit |
| **Assembly Process** | 1. Position split ring with pliers → 2. Thread through attachment point → 3. Ensure secure closure → 4. Test ring functionality (3 cycles) |
| **QC Checks** | 360° rotation test, pull test (gentle resistance), no sharp edges/burrs, print alignment |
| **Packing** | Custom Design Card + Seal King pouch OR Bubble Wrap → Small Box OR Pouch |
| **Storage** | Flat storage required |

#### FRIDGE MAGNETS

| Attribute | Specification |
|-----------|--------------|
| **Materials** | Fiber Print OR Birch Wood |
| **Print Type** | Single side print/engraving |
| **Attachment** | Round Magnet Backing (Neodymium grade) — sizes 15mm, 20mm diameter |
| **Adhesive** | Fevi Quick Extra Thick Glue |
| **Assembly Complexity** | Moderate |
| **Assembly Time Target** | 3–4 minutes per unit |
| **Assembly Process** | 1. Apply Fevi Quick Extra Thick → 2. Position round magnet backing → 3. Apply firm pressure (30 seconds) → 4. Allow curing time (2 minutes) |
| **QC Checks** | Magnetic strength test, adhesion security, no glue overflow/residue, surface finish |
| **Packing** | Magnetic Info Card + Protective Sleeve → Non-Magnetic Box OR Pouch |
| **Special Handling** | Keep separated from metal objects; use non-magnetic packaging |

#### LAPEL PINS

| Attribute | Specification |
|-----------|--------------|
| **Materials** | Fiber Print OR Birch Wood |
| **Print Type** | Single side print/engraving |
| **Attachment** | Rubber clutch backing OR Magnetic backing — 50 each type stocked |
| **Adhesive** | Fevi Quick Extra Thick Glue |
| **Assembly Complexity** | Moderate |
| **Assembly Time Target** | 2–3 minutes per unit |
| **Assembly Process** | 1. Apply precise amount of glue → 2. Position backing with alignment → 3. Apply firm pressure (45 seconds) → 4. Check bond during curing |
| **QC Checks** | Backing attachment security, design visibility/alignment, no adhesive overflow, professional finish |
| **Packing** | Backing Card with Logo + Small Protective Pouch → Small Box OR Padded Pouch |
| **Special Handling** | Sharp point protection |

#### ENAMEL PRODUCTS

| Attribute | Specification |
|-----------|--------------|
| **Assembly Complexity** | Varies (some QC only, some backing attachment) |
| **QC Checks** | Enamel quality, rim protection, print durability |
| **Packing** | Premium packaging with enamel-specific protection |

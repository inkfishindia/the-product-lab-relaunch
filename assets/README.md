# Assets Directory

Brand assets for The Product Lab relaunch. All visual assets live here.

## Structure

```
assets/
|-- brand/          -> Logo files, color swatches, typography reference, brand guidelines
|-- product/        -> Product photography (hero shots, lifestyle, catalog)
|-- social/         -> Instagram templates, story frames, reel thumbnails
`-- creative/       -> Campaign assets, launch materials, seeding kit
```

## Brand System

D-015 was superseded by D-018 and D-021. The current direction is opinion-first, organised chaos, colour from products, and a light + bold commerce experience.

### Current Direction

| Decision | Status | Meaning |
|---|---|---|

| D-021 | Active | Brand visual experience locked as opinion-first, organised chaos, colour from products |
| D-015 | Superseded | Darkroom + Type Pressure is no longer the primary direction |

### Core Palette

| Token | Hex | Use |
|-------|-----|-----|
| `--tpl-cream` | `#F5F0EB` | Primary background |
| `--tpl-black` | `#1A1A1A` | Primary text / high-contrast UI |
| `--tpl-red` | `#E63B2E` | Signal accent, CTAs |
| `--tpl-yellow` | `#F2D024` | Electric accent, highlights |

### Typography

| Role | Font | Weight |
|------|------|--------|
| Display / Headlines | Barlow Condensed | 700-900 |
| Body / UI | Inter | 400, 500, 600 |

### Logo Expressions

Three approved expressions:
1. **Wordmark** - "THE PRODUCT LAB" full text
2. **TPL Monogram** - Compact mark for small applications
3. **Tagline Lockup** - Wordmark + "Wear your opinion."

### Visual Direction

Opinion-first and product-led. The site should feel bold, direct, collectible, and commercially clear. Product colour and sticker artwork should carry the energy; UI structure should stay usable and scannable.

## File Naming Convention

```
[type]-[descriptor]-[variant]-[size].[ext]

Examples:
logo-wordmark-white-svg.svg
logo-monogram-dark-512.png
product-cardsticker-bataangle-hero.jpg
social-post-template-product-1080.png
```

# Social Media Templates Brief

> For Canva execution. Use brand tokens from assets/brand/brand-tokens.md.

## Instagram Feed Templates (1080×1080)

### Template 1: Product Feature
```
[Background: #1A1A1A]
[Product photo centered, 60% canvas]
[Product name: Barlow Condensed 900, #F5F0EB, bottom-left]
[Price: Inter 600, #F2D024, below name]
[TPL wordmark: top-right, small, #F5F0EB]
```

### Template 2: Opinion/Attitude Post
```
[Background: #E63B2E or #F2D024]
[Bold text: Barlow Condensed 900, full-bleed type]
[Text color: contrasting (white on red, black on yellow)]
[No imagery — type-only]
[TPL monogram: bottom-right, small]
```

### Template 3: Artist Feature
```
[Background: #1A1A1A]
[Artist work: left 60%]
[Artist name: Barlow Condensed 700, #F5F0EB, right panel]
["Artist at TPL": Inter 400, #6B6B6B, below name]
[TPL wordmark: bottom]
```

### Template 4: Launch Countdown
```
[Background: #1A1A1A]
[Large number: Barlow Condensed 900, #F2D024, center]
["Days until drop": Inter 500, #F5F0EB, below]
[TPL wordmark: bottom-center]
```

## Caption Framework

See `artifacts/phase-3/copy-system.md` for voice rules.

Standard structure:
```
[Hook line — 1 sentence, opinionated]
[Product/context detail — 1-2 sentences]
.
.
[CTA — direct, no exclamation marks]
[Line break]
[Hashtags — 5 max, relevant only]
```

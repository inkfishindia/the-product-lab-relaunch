1. **WooCommerce API requires auth** — the `wc/v3/products` endpoint returns 401. Only the public WP REST API (`wp/v2/product`) was accessible, which doesn't include price, stock, or SKU data.

2. **Yoast SEO is configured** but product-level meta descriptions are mostly empty — only `og:title` and `og:description` are populated.

3. **Twitter card meta is entirely null** for all products — potential optimization opportunity.

4. **Divi page builder** is used extensively — most products have `_et_pb_use_builder` and `_et_pb_old_content` meta fields.

5. **63 product URLs lack images** — these are likely duplicate/variant listings that should be cleaned up or canonicalized.

6. **The site has both old (2024) and new (2025) product lines** — the 2025 designs use a more consistent naming convention (`TPL-{design}.png`) and higher-quality artwork.

7. **Sell Your Art / Internship / Join the Team pages** suggest the brand is actively building community and team.

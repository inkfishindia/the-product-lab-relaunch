# Catalogs

Product catalog sources, cleaned exports, SKU references, and inventory tracking live here.

## Use This For

- Product catalog CSV/XLSX files
- Cleaned product data
- SKU and inventory references
- Catalog transformation scripts that have not yet moved into a package

## Do Not Use This For

- Live storefront components
- Backend commerce logic
- Raw WooCommerce/Fynd exports that are only historical references

Future home: `commerce/catalog/` or `packages/catalog/` after the workspace migration.

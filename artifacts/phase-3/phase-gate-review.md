## Key Decisions Locked in Phase 3

1. **Visual system:** Darkroom — off-black (#1A1A1A) base, warm white (#F5F0EB) text, signal red (#E63B2E) and electric yellow (#F2D024) accents.
2. **Typography:** Barlow Condensed (headings, condensed, uppercase) + Inter (body).
3. **Logo direction:** Three expressions — full wordmark, TPL monogram, tagline lockup. Typographic only, no symbol.
4. **Copy voice:** Five rules locked (have an opinion, group-chat register, edge on the humor, name the person not the product, short enough to forward).
5. **Site structure:** 20+ URL paths. Drop culture mechanics built in (/the-drop/). Gifting hub with warm context override.
6. **Gifting exception:** Warm surface (#F5F0EB bg) used only on /gifts/, /bundles/, and checkout — not a second theme.
7. **Product photography brief:** Dark posterboard, directional desk lamp, smartphone camera. No studio required.

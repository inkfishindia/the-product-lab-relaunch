## Done Checklist

Before marking copy as complete:

- [ ] Identity hook names the person, not the product
- [ ] No banned words: quirky, unique, fun, curated, explore, premium, vibrant, everyday, special, amazing
- [ ] WhatsApp line <15 words, works without context
- [ ] Specs are factual list, not persuasive paragraph
- [ ] Price matches D-006 pricing table
- [ ] Artist attribution filled (or marked "in-house" if not applicable)
- [ ] Forwarding test: paste the WhatsApp line into a chat — does it make sense?

## How To Use This Document

This is a ready-to-paste prompt library for Google Gemini (Imagen). Each prompt is designed to output assets that match the locked Darkroom + Type Pressure visual system (D-015).

**Before generating:**
1. Copy the exact prompt text (everything inside the code block)
2. Paste into Gemini
3. Review output against the brand rules below
4. Regenerate with tweaks if needed — common adjustments noted per section

**Brand Rules Every Output Must Follow:**

| Rule | Spec | Source |
|------|------|--------|
| Background | Dark — matte black, dark concrete, dark fabric (#1A1A1A zone) | Sean D-015 |
| Lighting | Single directional source, 45° angle, product is brightest element in frame | Sean |
| Color accents | Signal red (#E63B2E), Electric yellow (#F2D024) — used sparingly | Sean |
| Text style | If type is present: condensed, bold, all-caps, white on dark | Sean (Type Pressure hybrid) |
| Photography | No pastels. No white backgrounds. No full-body models. Tight crops. Moody indoor light. | Sean |
| Never | No illustrations. No mascots. No decorative elements. No filters. No HDR look. | Sean |
| Mood | A zine printed at midnight. High contrast. Precise darkness with sharp color punctuation. | Sean |

#### P-01: Enamel Pin — Hero Shot (Dark Surface)

```
Product photography of a small enamel lapel pin with bold text design, placed on a matte black posterboard surface. Single warm directional light from upper-left at 45 degrees creating a defined shadow falling to the lower-right. The pin is the brightest object in the frame — metallic edges catching the light, enamel colors vivid against the dark background. Shot from a 45-degree overhead angle showing the full face of the pin and slight depth of the metal edge. Shallow depth of field, background falls to pure darkness at the edges. No props, no hands, no other objects. Moody, editorial feel — like a product shot from a design magazine. High resolution, sharp focus on the pin text and enamel detail.
```

## PROMPT ENGINEERING TIPS FOR GEMINI

### Getting Better Results

1. **If the output is too bright:** Add "very dark environment, low-key lighting, the product is the only bright element" to the end of the prompt.

2. **If products look AI-generated:** Add "photorealistic, shot on a real camera, natural imperfections in lighting" to counteract the synthetic look.

3. **If backgrounds aren't dark enough:** Specify "background is #1A1A1A off-black, NOT gray, NOT charcoal — near-black with warm undertone."

4. **If text in images looks wrong:** Gemini can struggle with text generation. For typographic posts (S-01, S-03, etc.), consider generating only the photography element and adding text in Canva afterward.

5. **If the mood is too polished:** Add "slight film grain, not digitally perfect, editorial magazine photography style" to avoid the hyper-clean AI aesthetic.

6. **For consistency across a batch:** Start each prompt in a session with: "Continuing the same visual style as previous images: dark matte backgrounds, single warm directional light from upper-left, high contrast, editorial product photography."

7. **Aspect ratios:** Specify explicitly: "square 1:1 composition" for Instagram feed, "9:16 vertical composition" for Stories/Reels, "16:9 landscape" for website banners.

### What Gemini Handles Well
- Photorealistic product photography on dark surfaces
- Moody atmospheric lighting
- Close-up and macro detail shots
- Flat lay compositions
- Lifestyle context (hands holding objects, objects on surfaces)

### What May Need Manual Fix (Use Canva After)
- Any text overlays (brand name, headlines, price tags)
- Exact hex color matching
- Precise logo placement
- Consistent font rendering across images

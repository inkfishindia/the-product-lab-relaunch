# Prototypes

Experimental screens, HTML prototypes, and design proofs live here until promoted.

## Use This For

- Static HTML prototypes
- Framer/Vite experiments
- Roadmap, dashboard, and concept screens

## Do Not Use This For

- Production storefront routes
- Production admin screens
- Final design system documentation

Promote into an app only after the prototype has an owner, route plan, and build path.

## How to Use This Document

Each agent gets a briefing package: a prioritized list of files to read before starting work. Files are split into:
- **ALWAYS READ** = required for every agent, every session
- **READ FOR ROLE** = specific to your function
- **REFERENCE** = read only if your task requires it

**All agents read these first, regardless of role:**
| File | Why |

| `decisions/decision-log.md` (last 5 entries) | What's been decided — do not relitigate |

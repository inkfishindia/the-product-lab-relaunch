## Rules

- Do not duplicate full artifacts here if they already live in `../artifacts/`.
- Link to app docs instead of copying app implementation details.
- Keep file descriptions current when adding or superseding documents.
- Prefer short, retrievable source-of-truth docs over broad narrative dumps.

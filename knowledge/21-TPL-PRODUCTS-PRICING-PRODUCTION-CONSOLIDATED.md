### Labor
₹3/piece base (₹1 per process step: base, print, attachment)

### Unit Cost Formula

```
Unit Cost = Material Cost + (Print Cost ÷ pieces per sheet) + Attachment + Packaging + Labor

Where:
  Material Cost = (size_inches² ÷ 144) × material_rate × wastage_factor
  Pieces per Sheet = floor(sheet_width_mm ÷ (product_width_mm + 6mm)) × floor(sheet_height_mm ÷ (product_height_mm + 6mm))
  Bleed = 3mm per side = 6mm total
```

### Worked Example: 1"×1" Lapel Pin (Acrylic, UV A4, Rubber Back, Plain Card)

```
Cutting size: 31.4mm × 31.4mm
Pieces per A4: 6 × 9 = 54 pieces
Material: (1/144) × 90 = ₹0.625
Print: 200/54 = ₹3.70
Attachment: ₹6
Packaging: ₹5
Labor: ₹3
UNIT COST: ≈₹18.33
```

## PART 4 — PRODUCTION MODEL

### Core Model
- **100% Print-on-Demand (POD) / Just-in-Time (JIT)**
- Everything made to order — zero pre-built stock for D2C
- No MOQ for individual customers
- Maximum stock policy: 45 days (applied to components, not finished goods)

### Order-to-Fulfillment Timeline

| Stage | Time Window | Current Performance |
|-------|-------------|---------------------|
| Processing | 0–6 hours | Receive, validate, allocate |
| Production | 6–18 hours | Print, manufacture |
| QC | 18–20 hours | Inspection |
| Packed | 20–22 hours | Packaging and labeling |
| Shipped | 22–24 hours | Handoff to logistics |
| **Total** | **<24 hours target** | **18h avg actual** |

### Dispatch Priority System (Bin Color Coding)

### Routing Principle
Factory routing is product-based, not client-based. All keychains go to AC regardless of which client ordered them. This enables batch production efficiency. Factories return mixed batches sorted by design/product type; the Ops Manager then sorts into client-specific and order-specific bins.
